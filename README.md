# Matmatics
Matmatics is a Java Program which is designed to be a user friendly GUI program that is able to calculate and perform tasks around what is deemed "interesting math concepts".

It's purpose is to help make confusing math that may or may not be important more accessible. May or may not include tutorials and explanations in future.

## Description
Matmatics is intented to be a .exe file that launches a simple program that you can access multiple menus, each of them doing different math concepts. Each menu will be unique and provide calculations based around different maths that the creators deem "interesting". It is made in hopes to make these interesting things more accessible and easy to use as a calculator.

In the future, the program may or may not include in-depth tutorials and explanations of the concepts provided, as well as just work as an ordinary calculator.

## Getting Started

### Dependencies
The program should only need Java installed. It was programmed in Version *"24.0.1" 2025-04-15*. Earlier versions are unknown whether they would work or not.

### Installing
Download all files provided (Except for the README.md) and install them into one folder.

### Executing Program
Install all files within one folder, then use the built-in Run tool in Visual Studio and execute the *Runner.java* file.
